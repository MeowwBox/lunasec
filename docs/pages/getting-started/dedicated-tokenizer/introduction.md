# Setting up LunaSec

For now, LunaSec is designed to work with a system that uses React on the frontend and Express or Apollo-Graphql on the backend.  
Other frontend frameworks are on the development roadmap, including Vue, and serverside Java support is in alpha.  

To see a full demonstration while you follow this guide, you can look at our [demo app's source].  

We will be installing a couple of NPM modules:
On the 
* Dedicated Tokenizer server
* AWS (we recommend localstack)
